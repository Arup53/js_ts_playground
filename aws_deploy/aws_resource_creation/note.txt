------------ PERMISSION -------- 
1. GO TO LAMBDA CONFIGURE AND CLICK PERMISSION
2. CLICK EXECUTION ROLE NAME --> GO TO THE JSON EDITOR AND EDIT FOR SNS PUSHING FROM LAMBDA AND PASTE THE FOLLOWING
 {
            "Effect": "Allow",
            "Action": "sns:Publish",
            "Resource": "arn:aws:sns:ap-southeast-1:325899476249:customerIoTest"
        }
3. FOR LAMBDA WITH SQS SUBSCRIBER ADD PERMISSION FIRST BEFORE ADDING trigger
{
			"Effect": "Allow",
			"Action": [
				"sqs:ReceiveMessage",
				"sqs:DeleteMessage",
				"sqs:GetQueueAttributes"
			],
			"Resource": "arn:aws:sqs:ap-southeast-1:325899476249:customerIoQueue"
		}
